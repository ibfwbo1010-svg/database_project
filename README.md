# Vietnam K-POP Idol Trend Service

## 📖 프로젝트 개요
본 프로젝트는 **베트남 한류 아이돌 커뮤니티 데이터와 인기도 데이터를 기반으로**  
베트남에서 인기 있는 K-POP 아이돌 트렌드를 분석·시각화하는 웹 서비스입니다.

월별 베트남 커뮤니티 키워드 데이터를 활용하여  
아이돌별 관심도와 인기도를 함께 보여주는  
**데이터 기반 한류 트렌드 분석 서비스**를 목표로 합니다.

---

## 🎯 주요 기능
- 📅 **월별 아이돌 트렌드 분석**
  - 월 선택을 통해 베트남 커뮤니티 관심도가 반영된 아이돌 트렌드 확인

- ⭐ **아이돌 개인 페이지**
  - 아이돌 기본 정보 (그룹, 소속사, 데뷔년도)
  - 2019년 월별 인기도 테이블 제공
  - 베트남 커뮤니티 주요 키워드 분석 결과 제공

- 🔥 **베트남 커뮤니티 TOP 키워드**
  - 월별 베트남 커뮤니티에서 가장 많이 언급된 키워드 확인

- 📝 **게시글 & 댓글 기능**
  - 간단한 커뮤니티 게시글 작성
  - 댓글 작성 기능 제공

---

## 🗂 데이터 구성
- **아이돌 정보 데이터**
  - TOP 30+ 한류 아이돌 (개인 단위)

- **아이돌 월별 인기도 데이터**
  - 2019년 기준 월별 인기도 (SQLite DB)

- **베트남 커뮤니티 키워드 데이터**
  - 베트남 커뮤니티 내 아이돌 관련 키워드 언급 빈도 데이터
  - 월별 키워드 트렌드 분석에 활용

---

## 🛠 기술 스택
- **Backend**: Python, Flask
- **Database**: SQLite3
- **Frontend**: HTML, CSS (Jinja2 Template)
- **Version Control**: Git / GitHub
